A web app that processes emails and PDF attachments across multiple Microsoft 365 mailboxes. Attachments are stored in Azure Blob Storage and passed to OpenAI GPT and Azure Document Intelligence for structured data extraction using enforced JSON schemas.
Built with Logic Apps and direct file handling for scalability and high-accuracy OCR + AI processing.
